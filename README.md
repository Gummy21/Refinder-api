# Reinder-api
The back-end of Refinder

## General info

The back-end of the Refinder app, built using NodeJS, Axios and Express.

## Technologies
Created with:
* NodeJS: v14
* Axios: 0.21.1
* Express: 4.17.1

## Setup 
To run this project, install it locally using npm:
```
$ cd ../refinder-api
$ npm install
$ npm start
